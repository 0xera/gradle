User Guide

PLEASE NOTE! The user guide often refers to Gradle's javadoc to get detail information. Gradle is written partly in Groovy. Groovy does not provide a production ready way to generate API documentation. For the Groovy parts of Gradle you have to directly look into the source code to get the information you want. You might use Fisheye for this. The source code is also part of the Gradle distribution. - With our next release we will provide a complete API documentation.
